# ship.md

Claude Code skills for shipping software with confidence.

## Skills

### `/ship` — Full-Cycle Workflow

9-phase pipeline for non-trivial features and fixes: interview → explore → plan → implement → verify → edge cases → simplify → security review → final verify.

Uses the strongest available model for planning and autonomous goal loops for quality gates.

### `/ship-simple` — Streamlined Workflow

4-phase pipeline for straightforward tasks where requirements are already clear: explore → plan → implement → verify.

## Install

Add to your Claude Code project:

```
cp -r .claude/ your-project/.claude/
```

Or install via the [A Major Skills](https://github.com/amajorai/skills) collection.
